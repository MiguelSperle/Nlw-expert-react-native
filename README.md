### How to Clone this Repository


```⌨ Clone the repository```

```
git clone https://github.com/MiguelSperle/nlw-expert-react-native.git
```

```📂 Access at folder```

```
cd nlw-expert-react-native
```

```📡 Install dependencies```

```
npm install or yarn install
```

```⭐ Start the project```

```
npx expo start
```
